Como funcionam os websites?

Os nomes dos websites que conhecemos são na verdade como nomes fantasias de empresas. Quando acessamos um site pela aba de busca do navegador, estamos na verdade fazendo uma busca, enviando o "nome fantasia" e recebendo o endereço IP do servidor que hospeda o site que buscamos.

Os nomes fantasias (www.google.com, www.youtube.com e etc) são oferecidos pelo DNS(Domain Name System) para facilitar o acesso aos websites. É muito mais fácil memorizar um nome como "www.amazon.com.br" do que um endereço IP, composto por 4 números separados por pontos como "127.102.079.100".

Após receber o endereço IP, nos conectamos ao servidor e recebemos arquivos que compõem o site. Dentre estes arquivos temos 3 principais: HTML, CSS e Javascript.

O HTML(Linguagem de Marcação de Hipertexto) é um documento de hipertexto responsável pela estruturação do site. Ele define os elementos presentes na página como imagens, textos, audios, vídeos, menus e etc. Além disso ele também fornece algumas informações para o navegador e mecanismos de busca.

O CSS(Folha de Estilo em Cascata) é o responsável pelo estilo da página. Ele altera as propriedades visuais dos elementos do HTML como: cores, fontes, margens, posição, background.

O Javascript é o responsável pela lógica de programação. Ele é uma linguagem de programação baseada em scripts executada de forma nativa nos navegadores. Dentre suas funções temos alterações de elementos em páginas dinâmicas.
